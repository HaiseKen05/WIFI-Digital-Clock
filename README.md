# WIFI-Digital-Clock
A digital clock using ESP32, any display module thats I2C still going to add more features soon

# Wiring Diagram 
LCD I2C 20x4
- GND - GND
- VCC - VCC
- SDA - D21 
- SCK - D22

# Whats New? 
- Added the basic clocks for the following display modules: ILI9341, SSD1309, SSD1306 and SH110X 
- Added Clock UI for the ILI9341 and SH110X
- Remade/Refactored the Clock UI for the SH110X

# Q: Will be there be more features coming?
